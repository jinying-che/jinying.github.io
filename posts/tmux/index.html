Tmux Cheat Sheet

The basic usage of Tmux, personally, I think Tmux makes more sense in server side than local.

Session

  • new session

tmux new -s name

  • show session

tmux ls

  • attach session with name

tmux a -t name

  • detach from session

ctrl+b then d

Panes

  • split pane horizontally

ctrl+b then %

  • split pane vertically

ctrl+b then "

View in tmux

ctrl+b then [
